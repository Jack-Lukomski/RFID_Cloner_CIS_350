This file contains JavaScript code for interacting with a cloner device via Bluetooth. It includes functions for connecting to the cloner, sending and receiving data, setting up notifications, and controlling the cloner device. It also provides utility functions for storing and retrieving RFID codes in local storage and updating the user interface.
